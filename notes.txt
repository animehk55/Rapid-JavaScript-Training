any tag written in <script> tags works immediately.
---------------------------------------------------------------------------------------
in we write anything in source-file that src="app.js" then it will ignore the other code written there 

<script type="text/javascript" src="appjs">console.log("Helllo")</script>
here hello will not be printed, it will be ignored, rather files written in app.js will be printed.
---------------------------------------------------------------------------------------
XML tag : in it we can not end with finall tag.
        <input />
html tag: in it we have to always end with finall tag.
        <script></script>
---------------------------------------------------------------------------------------
  "defer" will be written to execute the files afterwards, in src tag:
  <script defer src = "app.js"></script>
---------------------------------------------------------------------------------------
<noscript>No JS available</noscript>
is used to represent that there is not JS available.
